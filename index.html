LS(1)

Name

ls


list directory contents
└─$ ls
example_audio_file.mp3 example_backup_file~ example_compressed_file.tgz example_dir example_exec_file.sh example_fifo_file example_file 'example file with spaces' example_image_file.jpg example_link_file example_socket_file example_text_file.txt

Synopsis

Usage


ls [OPTION]... [FILE]...
└─$ ls --help
Usage: ls [OPTION]... [FILE]...

Description

List information about the FILEs (the current directory by
default). Sort entries alphabetically if none of -cftuvSUX nor
--sort is specified.

Mandatory arguments to long options are mandatory for short
options too.
└─$ ls --help
List information about the FILEs (the current directory by default). Sort entries alphabetically if none of -cftuvSUX nor --sort is specified.

change the default of using modification times; access
time (-u): atime, access, use; change time (-c): ctime,
status; birth time: birth, creation;

with -l, WORD determines which time to show; with
--sort=time, sort by WORD (newest first)
